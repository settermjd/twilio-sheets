## Twilio / Google Sheets Integration

This is a small project showing how to integrate Order Desk, Twilio, and Google Sheets.

## Prerequisites

To use the package, you'll need the following:

- [PHP][php] 8.4
- [Composer][getcomposer] installed globally
- A Twilio account (either free or paid).
  [Sign up now][try-twilio] if you don't have one already.
- A Google account
- An [Order Desk][order-desk] account
